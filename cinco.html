Ejecución y Muestra del error 403 Forbidden


    Luego de haber realizado todos los pasos anteriores es posible que de error 403.14 Forbidden (El servidor web está configurado para no mostrar una lista los contenidos de este directorio).


    Para solucionar el problema se debe realizar:

1. Abra el Administrador de IIS.

2. En la vista Características, haga doble clic en Examen de directorios.

3. En la página Examen de directorios del panel Acciones, haga clic en Habilitar.
